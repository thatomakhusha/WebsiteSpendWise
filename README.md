# SpendWise – Personal Finance Tracker

SpendWise is a web-based personal finance management application designed to help users track income, expenses, and budgets in a structured and visual way. The application focuses on real-world financial tracking features such as categorisation, monthly summaries, and budgeting.

This project is currently in active development and is being expanded into a full-stack application using Firebase.

---

##  Features

- User authentication (Firebase Authentication)
- Dashboard for income, expenses, savings, and balance tracking
- Add, edit, and delete transactions
- Category-based income and expense tracking
- Monthly filtering system for financial data
- Budgeting system per category
- Spending analytics and summaries
- Cloud data storage using Firebase Firestore (migration from localStorage)

---

##  Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Firebase Authentication
- Firebase Firestore
